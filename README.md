# React + Apollo GraphQL Playground

A small playground project built to learn GraphQL, Apollo Server, Apollo Client, queries, mutations, caching, and React integration.

## Tech Stack

- React
- Vite
- Apollo Client
- Apollo Server
- GraphQL

## Features

- Fetch all users
- Fetch user by ID
- Create new users with GraphQL mutations
- Apollo cache refetching
- Local GraphQL server

## Goal

This project was focused purely on understanding the data flow and backend/frontend communication under the hood rather than styling/UI.
